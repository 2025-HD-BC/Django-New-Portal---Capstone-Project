GitHub Repository Setup Instructions
=========================================

Once you create a GitHub repository, use these commands to push your code:

1. Add the remote repository (replace YOUR_USERNAME and YOUR_REPO_NAME):
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git

2. Push your code to GitHub:
   git branch -M main
   git push -u origin main

3. Your repository will be available at:
   https://github.com/YOUR_USERNAME/YOUR_REPO_NAME

Example commands if your GitHub username is "johndoe" and repo is "django-news-portal":
   git remote add origin https://github.com/johndoe/django-news-portal.git
   git branch -M main
   git push -u origin main

Then your link would be: https://github.com/johndoe/django-news-portal
