Django News Portal Documentation
=================================

Welcome to the Django News Portal documentation. This application provides a comprehensive
news management system with role-based access control for readers, journalists, editors,
and publishers.

Features
--------

* **User Management**: Multi-role user system with custom authentication
* **Article Management**: Create, review, approve, and publish news articles
* **Publisher System**: Organize content by publishing organizations
* **Newsletter System**: Create and distribute newsletters
* **Subscription System**: Allow readers to follow publishers and journalists
* **REST API**: Comprehensive API for external integrations
